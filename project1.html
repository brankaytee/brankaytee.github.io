ORGANIC MODEL - Alien Character

Programs Used

ZBrush, Maya

2019

The story behind this project was to create a low poly alien character to fit a futuristic space environment where the characterâ€™s role is a galactic mercenary.

It was my first time using ZBrush to sculpt, which made it difficult to navigate the interface and understand the tools. To overcome this, I practiced sculpting separate similar shapes that replicated the structural elements of the character. Through this practice I was able to sculpt the final character more efficiently with a stronger understanding of ZBrush.

Process

Sculpting the character from a sphere in ZBrush

ZRemesher to retopologize triangles into squares

Modelled armour based off topology of character in Maya

Added more subdivisions to mesh to create smoothness
